About Infinite Math

Welcome to Infinite Math, an interactive visualization of mathematical equations in 3D space. This project aims to make mathematical concepts more engaging and accessible through interactive digital art.

Each rotating cube in the spiral represents a different mathematical equation, carefully selected for both their mathematical significance and aesthetic appeal. By clicking on any cube, you can explore the equation in detail.

This project coombines:

  • Three.js for 3D visualization
  • LaTeX for mathematical notation
  • Interactive web technologies

Created by Zack ElMetennani, this project is an ongoing exploration of math, art, and technology.
